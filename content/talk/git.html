GIT als freies, verteiltes, sicheres, performantes und umfangreiches Versionsverwaltungssystem ist heute für viele Projekte die erste Wahl. Dieser Vortrag gibt eine Einführung in die Funktionsweise, die wichtigsten Befehle und die Architektur von GIT. Im Anschluss folgt ein kleiner Workshop, bei welchem die Teilnehmer git auf ihrem Notebook installieren können, um anschließend die Inhalte vom Vortrag selbst anzuwenden.
